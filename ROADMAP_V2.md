从 V2 开始，玄风门升级定位为 **玄风命理平台**。

## 平台架构

```
玄风命理平台
    │
    ├── 推演层 (GPT-4o / Gemini / Edge)
    │
    ├── 规则引擎层 (八字规则 / 风水规则 / 奇门规则 / 六壬规则 / 紫微规则)
    │
    ├── 知识库层 (经典文献 / 案例 / 学派)
    │
    ├── 推理引擎层 (Qi Engine / 概率引擎 / 决策引擎)
    │
    └── 应用层 (八字 / 风水 / 合盘 / 起名 / 择日 / 奇门 / 六壬 / 紫微)
```

所有命理模块共享同一套基础设施：
- 统一的 推演 调用层（三引擎切换 + Token 控制）
- 统一的缓存层（5 级缓存）
- 统一的安全层（CSP + Rate Limit + Auth）
- 统一的支付层（微信/支付宝/Stripe）
- 统一的监控层（Sentry + GA4 + Clarity）
- 统一的会员体系（Free/Pro/Master）
