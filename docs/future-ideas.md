## 想法清单

### IDEA-001：AI 命例反查引擎

- **标题**：AI 命例反查引擎
- **背景**：用户给出已知命理结论（如"此人35岁破财"），反查符合的命例特征。
- **收益**：提升断命准确率参考库；商业差异化。
- **风险**：反查逻辑可能误判，需大量命例支撑。
- **优先级**：低
- **预计版本**：V5
- **负责人**：待定
- **状态**：Idea

---

### IDEA-002：紫微斗数排盘插件

- **标题**：紫微斗数排盘
- **背景**：玄风平台扩展到紫微斗数，复用 Rule Engine + Explain 体系。
- **收益**：平台能力扩展；商业化新模块。
- **风险**：紫微排盘算法复杂，需要平台升级（Plugin Architecture 已就绪）。
- **优先级**：中
- **预计版本**：V5
- **负责人**：待定
- **状态**：Idea

### IDEA-003：Dashboard 实时大屏可视化

- **标题**：Rule Statistics Dashboard 实时大屏
- **背景**：当前 Dashboard 为文本汇总，可升级为可视化图表。
- **收益**：运维可观测性提升；不影响准确率。
- **风险**：开发量大，可能分散算法精力。
- **优先级**：低
- **预计版本**：V4.8.1-pN（Patch，不升级基线）
- **负责人**：待定
- **状态**：Idea

### IDEA-004：真太阳时（均时差）校正

- **标题**：真太阳时排盘
- **背景**：基于出生经度计算真太阳时，修正均时差。

- **优先级**：高（属于 P0-④ 范畴）
- **预计版本**：当前开发版本（P0-④）
- **负责人**：待定
